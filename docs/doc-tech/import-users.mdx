## Introduction

Afin d'accéder à l'application, il faut tout d'abord y être inscris

Dans le but d'en sécuriser l'accès, un administrateur autorisé est l'unique façon d'inscrire des individus

Pour se faire, ce dernier se connecte à l'application et se rend dans la page "Inscription" accessible depuis le menu supérieur

Il lui est alors possible d'inscrire des utilisateurs de deux différentes façons

---

## Import Via Formulaire

Dans le cadre d'une inscription individuelle, il est possible de passer par un formulaire

Ainsi, il suffit de renseigner les champs obligatoires pour créer un utilisateurs

Ces champs sont les suivants : 

1. Prénom
1. Nom
1. Email
1. Id dans l'entreprise (Optionnel)
1. Numéro de téléphone (Optionnel)

![img](/img/import-users_form.JPG)

Une fois rempli et envoyé, ce formulaire envoie les données au serveur et crée l'utilisateur

Il lui est associé à un mot de passe généré aléatoirement qui lui est envoyé par mail

Il lui suffit par la suite de se connecter à l'application où il sera redirigé pour changer son mot de passe
